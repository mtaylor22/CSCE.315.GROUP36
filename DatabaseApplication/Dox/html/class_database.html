void Database::add_table ( string  name,
Table *  table 
)

Add a table to the database.

Ownership of the table is permanently transferred to the database, and it will be destroyed when the database is destroyed. You MUST allocate the table with new. Do NOT destroy the table after adding it to the database!

Database db;
Table *table = new Table;
db.add_table("my_table", table);

Throws an InvalidOperationError if name already exists in the database.

Parameters
namewhat to call the table in the database
tablethe Table to be inserted into the database

Table* Database::query ( string  select,
string  from,
string  where 
)

Perform a query on the database. Here are some examples of possible queries:

myDatabase.query("name", "students", "gender = 'male'");
myDatabase.query("name", "students", "(gender = 'male' AND age > 21) OR age > 65");
myDatabase.query("name", "students", "gender = 'female' AND id IN good_students");
myDatabase.query("*", "students", "gpa >= ALL(good_student_gpas)");
myDatabase.query("id", "students", "gpa >= ANY(good_student_gpas) AND NOT (id IN good_students)");
myDatabase.query("*", "students", "EXISTS(good_students)");

Throws a TableDoesNotExistError if from does not exist. Throws a QuerySyntaxError if select or where have a syntax error.

Parameters
selectwhich columns to include in the returned Table
fromwhich table to query from
wherethe conditions for the query to match

A pointer to Table with all of the records that match the query
